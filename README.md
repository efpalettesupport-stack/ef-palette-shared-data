 # EF-Palette 配布シード                                                                                        
                                                            
  iOS アプリ **EF-Palette** のユーザーが、共通の選手データを取り込むための配布リポジトリです。                   
                                                            
  ## 使い方                                                                                                      
   
  EF-Palette アプリ内で:                                                                                         
                                                            
  1. 設定 → データ管理 → 「URL を入力して取得」
  2. 表示される 4つの「貼り付けボタン」 から取り込みたいファイルを選択
  3. 「取得して内容を確認」 → プレビュー → 「N人を取り込み」                                                     
                                                                                                                 
  ## 配布ファイル (snapshots/)                                                                                   
                                                                                                                 
  | ファイル | 内容 | 更新頻度 |                            
  |---|---|---|
  | `YYYY-MM-DD-<release-name>.json` | 個別配信(その回のみ) | 週2回(月 ・木 想定) |
  | `YYYY-MM.json` | 月毎まとめ | 月初に前月分作成 |                                                             
  | `YYYY.json` | 年累計版 | 個別配信のたびに更新 |

   ## raw URL の取得方法 (iPhone)                                                                                 
                                                                                                                 
  snapshots フォルダ内の好きなファイルを開いて:                                                                
                                                                                                                 
  1. 右上 「・・・」 メニュー → 「Raw file content」                                                             
  2. **View を 長押し** → 「リンクをコピー」                                                                   
  3. EF-Palette アプリ → 設定 → 「URL を入力して取得」 → 貼り付け                                                
                                                            
  または View をタップ → 表示された画面のアドレスバーから URL コピーでも OK。                            
                                                            
  主要ファイルは アプリ内にボタンをご用意しています。 
                                                                                                                 
                                                                                                                 
  ## データに含まれるもの
                                                                                                                 
  - 選手の名前 ・ポジション ・初期ステータス                                                                     
  - 国籍 ・クラブ ・リーグ
  - 初期スキル ・ポジ適性 ・プレースタイル ・個性                                                                
  - 初期ブースター                                          
  - アイコン(あれば)
                                                                                                                 
  含まれないもの: 育成プラン、 拡張スキル、 拡張ポジ適性、 育成スナップショット
                                                                                                                 
  ## 重複判定                                               

  「名前 + 26ステータス初期値」 が同じ選手は ⚠️  既登録 と表示されます。                                          
  名前が同じでも初期値が違えば **別カード** として扱われます。
